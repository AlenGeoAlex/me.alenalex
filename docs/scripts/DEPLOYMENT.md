# Deployment Scripts

These scripts are used to prepare the environment and assets for deployment.

- **`scripts/deployment/update-env-backend-hono.mjs`**: Prepares the environment for backend deployment. (Triggered by `pnpm deploy:prepare-env-backend-hono`).
- **`scripts/deployment/update-favicon-pack.js`**: Replaces the default favicon in the Angular app with the production version during deployment.
