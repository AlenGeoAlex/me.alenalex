# GitHub Secrets and Environment Management

Both workflows rely on GitHub Secrets for sensitive information.

## Secrets List

- **Cloudflare**: `CF_API_TOKEN`, `CF_ACCOUNT_ID`.
- **Backend Env**: `BACKEND_ENV_ALLOWED_HOST`, `BACKEND_ENV_R2_*`, etc. (See [Portfolio API Environment](../portfolio-api/ENVIRONMENT.md)).
- **Azure**: `AZURE_STATIC_WEB_APPS_API_TOKEN_*`.
